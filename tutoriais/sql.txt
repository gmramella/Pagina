Existem vários SGBDs que têm dialetos diferentes mas têm SQL padrão como semelhança entre eles.
Primeiro é necessário saber o que são: index, view
Em seguida é necessário aprender a usar as instruções SQL padrão.
Por fim, se aprende os dialetos (MYSQL, ORACLE, SQL SERVER, POSTGRESQL etc).

		CREATE TABLE

		ALTER VIEW
